# Markdown Site Showcase

This repository turns one canonical Markdown tree into multiple public reading
models. The content is intentionally plain so the comparison stays focused on
navigation, defaults, and maintenance cost instead of custom design work.

## Start Here

- Read the [Concepts Hub](./concepts/index.md) for the mental model.
- Open the [Workflows Hub](./workflows/index.md) for the editing loop.
- Use the [Topics Hub](./topics/index.md) to compare deployment choices.

## What To Compare

- How easy each site makes it to move through sections
- How much structure each default layout gives you
- Whether links feel supportive or central to the experience
